-----------------------------------------------------------------------------
This is the project is about a watersort game whereby a user has to sort
the colours in the given bottle. It has a class called 'MyArrayList' that 
can create and manipulate an array of objects. The class 'StackAsMyArrayList' 
creates and manipulates an array of objects as stacks. Finally, use the 
'mainWatersort' class to run and play the program.
